Tulis kesimpulan ujicoba skenario1:di dalam sebuah container grid, dapat di isi menggunakan grid lagi dengan menggunakan div dengan nama item_grid.
Tulis kesimpulan ujicoba skenario2:Dengan mengubah nilai properti display menjadi inline-grid, elemen akan tetap sejajar dengan elemen lain dalam satu baris jika memungkinkan, namun tetap dapat menggunakan grid untuk menata isinya secara lebih fleksibel.
Tulis kesimpulan ujicoba skenario3:Menggunakan properti grid-column-gap dan grid-row-gap dalam elemen grid memungkinkan pengaturan jarak antar kolom dan baris dengan fleksibel. grid-column-gap mengontrol jarak horizontal antar kolom, sedangkan grid-row-gap mengatur jarak vertikal antar baris.
Tulis kesimpulan ujicoba skenario4:Menggunakan properti grid-gap pada item grid berfungsi untuk memberikan jarak yang seragam antara kolom dan baris dalam sebuah grid. Ini membantu dalam meningkatkan keterbacaan dan estetika tata letak tanpa perlu mengatur margin secara manual pada setiap elemen.
